# 04 - Next.js App Test

Next.js 14 application for testing SSR deployments.

## How to Deploy

```
Template: Deploy from GitHub
GITHUB_REPO: lexxxx89/test_repo
SUBDIRECTORY: 04-nextjs-app
+ Custom ENV (optional):
  CUSTOM_VAR: test_value
  NEXT_PUBLIC_API_URL: https://api.example.com
```

## What it tests:
- Next.js 14 App Router
- Multi-stage Docker build
- SSR (Server-Side Rendering)
- Environment variables
- Build optimization
- Production mode

## Expected result:
Next.js page with green terminal styling showing environment info

## Build time:
~3-5 minutes (includes npm install and build)
